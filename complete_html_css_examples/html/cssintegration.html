Inline, Internal, and External CSS

1. Inline CSS

CSS applied directly inside element.

<p style="color:red;">Text</p>

This is inline CSS example

2. Internal CSS

CSS written inside <style> tag in head.

<style> p { color:blue; } </style>

This is internal CSS example

3. External CSS

CSS written in separate file.

<link rel="stylesheet" href="style.css">

External CSS is best method for real websites.

Comparison

  • Inline CSS: Used for single element
  • Internal CSS: Used for single page
  • External CSS: Used for multiple pages (best method)
