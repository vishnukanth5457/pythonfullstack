HTML File Paths Tutorial

Learn Absolute Path and Relative Path

1. Absolute Path

Absolute path is full URL including domain.

https://example.com/images/photo.jpg

Used for external files.

2. Relative Path

Relative path is location inside your project folder.

images/photo.jpg
Example structure:

project/
├── index.html
└── images/
    └── photo.jpg
<img src="images/photo.jpg">

3. Relative Path Types

Same folder:
photo.jpg

Subfolder:
images/photo.jpg

Parent folder:
../photo.jpg

4. When to Use

  • Absolute path → External resources
  • Relative path → Internal project files
