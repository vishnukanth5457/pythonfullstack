HTML Colors Tutorial

Learn different ways to use colors in HTML

1. Color Names

HTML supports predefined color names.

This is red color

This is blue color

This is green color

style="color:red"

2. HEX Colors

HEX colors use hexadecimal values.

This is HEX color #ff5733

This is HEX color #3498db

style="color:#ff5733"

3. RGB Colors

RGB stands for Red, Green, Blue.

RGB Red

RGB Green

RGB Blue

style="color:rgb(255,0,0)"

4. RGBA Colors

RGBA adds transparency (alpha).

RGBA Red with transparency
style="background-color:rgba(255,0,0,0.5)"

5. HSL Colors

HSL stands for Hue, Saturation, Lightness.

HSL Red

HSL Green

HSL Blue

style="color:hsl(0,100%,50%)"

6. HSLA Colors

HSLA adds transparency.

HSLA Blue with transparency
style="background-color:hsla(240,100%,50%,0.5)"
