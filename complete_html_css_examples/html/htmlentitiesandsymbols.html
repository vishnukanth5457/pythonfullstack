Display reserved characters and symbols

1. Non-breaking Space ( )

&nbsp;
Hello   World

2. Less Than (<)

&lt;
<h1>This is heading</h1>

3. Greater Than (>)

&gt;
10 > 5

4. Copyright Symbol (©)

&copy;
© 2026 My Website

5. Registered Symbol (®)

&reg;
MyBrand ®

Why Use Entities?

  • Display reserved characters
  • Prevent HTML errors
  • Show symbols properly
  • Improve readability
