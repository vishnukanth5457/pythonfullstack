HTML URL Encoding Tutorial

Learn how URLs encode spaces and special characters

1. What is URL Encoding?

URL encoding converts special characters into a format that can be safely used in URLs.

URLs can only contain standard ASCII characters.

2. Space Encoding

Space cannot be used in URL directly.

Space → %20
Wrong:
https://example.com/my file.html

Correct:
https://example.com/my%20file.html

3. Special Character Encoding

# → %23
& → %26
@ → %40
= → %3D
? → %3F
/ → %2F

4. Example with Link

<a href="my%20page.html">Open Page</a>
Example encoded link: my%20page.html

5. Why URL Encoding is Important?

  • Prevents URL errors
  • Allows special characters in URLs
  • Required for query parameters
  • Used in forms and APIs
