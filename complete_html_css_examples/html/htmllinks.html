HTML Links Tutorial

Learn how to create links in HTML

1. Anchor Tag (a)

The anchor tag is used to create links.

<a href="https://example.com">Visit</a>
Visit Google

2. Absolute URL

Full web address including https://

Wikipedia
href="https://www.wikipedia.org"

3. Relative URL

Link to page inside your project.

Go to About Page
href="about.html"

4. Target Attribute

Controls where link opens.

Open in new tab (_blank)

Open in same tab (_self)
target="_blank"

5. Email Link

Opens email client.

Send Email
href="mailto:test@example.com"

6. Phone Link

Used to make phone calls.

Call Now
href="tel:+911234567890"

7. Bookmark Link

Jump to specific section in page.

Go to Bookmark Section
href="#bookmark-section"

Bookmark Section

You jumped here using bookmark link.
