HTML Images and Media Tutorial

Learn Images, Audio, and Video in HTML

1. Image Tag (img)

Used to display images.

<img src="image.jpg" alt="description">

2. src Attribute

Specifies image location.

src="image.jpg"

3. alt Attribute

Alternative text if image cannot load.

alt="Image description"

4. width and height

Controls image size.

width="200" height="150"

5. Image Formats

Supported formats: JPG, PNG, SVG, WEBP


6. Audio Tag

<audio controls>

7. Video Tag

<video controls width="300">

8. source Tag

Used inside audio or video.

<source src="file.mp4" type="video/mp4">

9. controls Attribute

Shows play, pause, volume controls.

controls

10. autoplay Attribute

Starts media automatically.

autoplay

11. loop Attribute

Repeats media continuously.

loop
