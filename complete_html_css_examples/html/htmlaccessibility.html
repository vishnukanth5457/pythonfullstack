1. alt Attribute (Images)

Provides text description for screen readers.

<img src="image.jpg" alt="Description">

2. aria-label

Provides label for screen readers.

aria-label="Close Button"

3. role Attribute

Defines element purpose.

role="button"
Custom Button

4. Semantic HTML

Use meaningful tags.

header, nav, main, footer
