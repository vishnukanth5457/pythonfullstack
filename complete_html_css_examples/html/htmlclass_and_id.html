4. Key Differences

CLASS:
  • Used for multiple elements
  • Reusable
  • Defined using .className in CSS
  • Example: .student
ID:
  • Used for one unique element
  • Cannot reuse
  • Defined using #idName in CSS
  • Example: #main-title

5. Real-world Example

Class = Category (Many students)
