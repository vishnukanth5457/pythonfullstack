HTML Head Elements Tutorial

Learn all head elements

1. title Element

Defines webpage title shown in browser tab.

<title>Page Title</title>

2. meta Element

Provides metadata like charset and viewport.

<meta charset="UTF-8">

3. link Element

Used to link external CSS file.

<link rel="stylesheet" href="style.css">

4. style Element

Used to write internal CSS.

<style> CSS code </style>

5. script Element

Used to add JavaScript.

<script> JS code </script>

6. base Element

Defines base URL for all links.

<base href="https://example.com/">
