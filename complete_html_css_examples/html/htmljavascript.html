HTML JavaScript Tutorial

Script tag, Inline JS, External JS, DOM Basics

1. Script Tag

Script tag is used to add JavaScript.

<script> alert("Hello"); </script>

2. Inline JavaScript

JavaScript written inside HTML element.

onclick="alert('Hello')"

3. External JavaScript

JavaScript written in separate file.

<script src="script.js"></script>

External JS is best practice.

4. DOM Basics

This text will change

document.getElementById("demo").innerHTML

What is DOM?

  • DOM = Document Object Model
  • JavaScript can change HTML content
  • JavaScript can change CSS
  • JavaScript can respond to events
