HTML Iframe Tutorial

Learn how to embed webpages, YouTube, and Google Maps

1. Basic iframe

iframe is used to embed another webpage.

<iframe src="page.html"></iframe>

2. YouTube Embed

Embed YouTube video using iframe.

<iframe src="https://www.youtube.com/embed/videoID"></iframe>

3. Google Maps Embed

Embed Google Maps location.

4. iframe Attributes

  • src – URL of page
  • width – width of iframe
  • height – height of iframe
  • title – description
  • allowfullscreen – enable fullscreen video
