HTML Attributes Tutorial

Learn HTML Attributes with Examples

1. What are Attributes

Attributes provide additional information about HTML elements.

Syntax:

<tagname attribute="value">
Example:
<p title="This is tooltip">Text</p>

2. Global Attributes

Global attributes can be used on ANY HTML element.

Examples: id, class, style, title, hidden, contenteditable, draggable, data-*

3. id Attribute

id provides a unique identifier for an element.

<p id="para1">This is paragraph</p>

This paragraph has an ID

4. class Attribute

class is used to group multiple elements for CSS styling.

<p class="text">Text</p>

This paragraph uses class attribute

5. style Attribute

style is used to apply CSS directly to an element.

<p style="color:red;">Red Text</p>

This text is red

6. title Attribute

title shows tooltip when mouse hovers on element.

<p title="Tooltip text">Hover me</p>

Hover mouse here

7. hidden Attribute

hidden hides the element from the webpage.

<p hidden>Hidden text</p>

This text is hidden

Hidden element is not visible

8. contenteditable Attribute

contenteditable allows user to edit the content.

<p contenteditable="true">Edit me</p>

Click here and edit this text

9. draggable Attribute

draggable allows element to be dragged.

<p draggable="true">Drag me</p>

Drag this text

10. data Attribute

data-* is used to store custom data in HTML elements.

<p data-user="admin">User</p>

This element has custom data attribute
