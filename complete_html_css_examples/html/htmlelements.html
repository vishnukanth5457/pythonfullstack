HTML Elements Tutorial

Learn HTML Elements with Explanation

1. What is an HTML Element

An HTML element consists of an opening tag, content, and a closing tag. Example: <p>This is a paragraph</p>

2. Opening and Closing Tags

Opening tag starts the element. Closing tag ends the element. Example: <h1>Heading</h1>

3. Empty / Self-closing Elements

Empty elements do not have closing tags. Example: <br> creates a line break.

4. Nested Elements

This paragraph is inside a div element. This is called nesting (element inside another element).

5. html Element

The html element is the root element. All HTML code must be inside html.

6. head Element

The head element contains metadata like title and meta tags. It is not visible on the webpage.

7. title Element

The title element defines the page title. It appears in the browser tab.

8. meta Element

Meta element provides information about the webpage like character encoding and responsive settings. It is an empty element.

9. body Element

The body element contains all visible content like headings, paragraphs, images, etc.
