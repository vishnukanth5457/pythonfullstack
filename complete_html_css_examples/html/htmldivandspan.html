HTML Div and Span Tutorial

Learn Block Container and Inline Container

1. div : Block Container

div is a block element used to group large sections.

<div>Content</div>
This is a DIV container (takes full width)

2. span : Inline Container

span is an inline element used to style part of text.

<span>Text</span>
This is normal text and this is span text inside the line.

3. Real-world Example (Room and Sticker)

div = Room (big container)
span = Sticker (small item inside room)

This is a ROOM (DIV)

This is a STICKER (SPAN) inside the room

4. Combined Example

Welcome to my website

My name is John

5. Key Differences

DIV:
  • Block element
  • Takes full width
  • Used for layout
  • Like a room
SPAN:
  • Inline element
  • Takes only needed width
  • Used for styling text
  • Like a sticker
