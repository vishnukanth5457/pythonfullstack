# üßæ TABLE METADATA TAGS (SEMANTIC üî•)

----------------------------------------------------------------------------------------------------------------------

## 5Ô∏è‚É£ `<caption>` ‚Äî Table Title

Provides a **heading for the table**.

üìå Appears at top

## 6Ô∏è‚É£ `<thead>` ‚Äî Table Header Group


## 7Ô∏è‚É£ `<tbody>` ‚Äî Table Body

## 8Ô∏è‚É£ `<tfoot>` ‚Äî Table Footer

# üîÄ ROW & COLUMN SPANNING

## 9Ô∏è‚É£ `rowspan`

## üîü `colspan` (Important to know)
