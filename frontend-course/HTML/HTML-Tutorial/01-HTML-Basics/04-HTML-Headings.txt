HTML TEXT AND FORMATTING

Used to display and format textual content on a web page.

----------------------------------------------------------------------------------------------------------------------

PART A TEXT TAGS STRUCTURE

These tags define structure not appearance.

----------------------------------------------------------------------------------------------------------------------

HEADINGS TAGS

The h1 to h6 tags are used for titles and sub titles.

Example

<h1>Main Heading</h1>
<h2>Sub Heading</h2>
<h3>Section</h3>
<h4>Sub Section</h4>
<h5>Small Heading</h5>
<h6>Very Small Heading</h6>

Rules

The h1 tag represents the most important heading
The h6 tag represents the least important heading
Only one h1 tag should be used per page as a best practice

Analogy

Book title corresponds to h1
Chapter corresponds to h2
Topic corresponds to h3

----------------------------------------------------------------------------------------------------------------------

PARAGRAPH TAG

The p tag is used for blocks of text.

Example

<p>This is a paragraph.</p>
<p>This is another paragraph.</p>

The browser automatically adds spacing between paragraphs.

----------------------------------------------------------------------------------------------------------------------

SPAN TAG

The span tag is an inline container.

It is used to style or target part of a text.

Example

<p>This is <span style="color:red;">important</span> text.</p>

The span tag has no semantic meaning by itself
It is an inline element

Analogy

A sentence is like a paragraph
A word is like a span

----------------------------------------------------------------------------------------------------------------------

BR TAG

The br tag creates a line break.

Example

Address:<br>
Hyderabad<br>
India

The br tag is a void element
It is commonly used for poems and addresses

----------------------------------------------------------------------------------------------------------------------

HR TAG

The hr tag creates a horizontal rule.

Example

<hr>

In HTML5 the hr tag represents a thematic break

----------------------------------------------------------------------------------------------------------------------

PART B FORMATTING TAGS MEANING AND STYLE

These tags affect both appearance and meaning.

----------------------------------------------------------------------------------------------------------------------

B AND STRONG TAGS

The b tag is used for bold text for visual styling only.

Example

<b>Bold text</b>

The strong tag is used to mark important text.

Example

<strong>Important text</strong>

Screen readers emphasize strong text
Strong is preferred over b for accessibility

----------------------------------------------------------------------------------------------------------------------

I AND EM TAGS

The i tag is used for italic styling only.

Example

<i>Italic text</i>

The em tag is used to emphasize text semantically.

Example

<em>Emphasized text</em>

The em tag improves accessibility

----------------------------------------------------------------------------------------------------------------------

U TAG

The u tag underlines text.

Example

<u>Underlined text</u>

It is not recommended for links as it can be confusing

----------------------------------------------------------------------------------------------------------------------

MARK TAG

The mark tag highlights text.

Example

<p>This is <mark>highlighted</mark> text.</p>

It is commonly used in search results

----------------------------------------------------------------------------------------------------------------------

SMALL TAG

The small tag displays smaller text.

Example

<small>Terms and conditions apply</small>

It is used for disclaimers and notes

----------------------------------------------------------------------------------------------------------------------

DEL TAG

The del tag represents deleted or removed text.

Example

<p>Price: <del>‚Çπ1000</del> ‚Çπ800</p>

----------------------------------------------------------------------------------------------------------------------

INS TAG

The ins tag represents inserted text.

Example

<p>Offer valid till <ins>Sunday</ins></p>

Inserted text is usually underlined

----------------------------------------------------------------------------------------------------------------------

SUB TAG

The sub tag is used for subscript text.

Example

H<sub>2</sub>O

It is commonly used in chemistry and mathematics

----------------------------------------------------------------------------------------------------------------------

SUP TAG

The sup tag is used for superscript text.

Example

10<sup>2</sup>

It is used for powers and footnotes
