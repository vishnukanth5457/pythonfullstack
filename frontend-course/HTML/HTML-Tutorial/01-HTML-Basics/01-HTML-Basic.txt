----------------------------------------------------------------------------------------------------------------------

WHAT IS HTML

HTML stands for HyperText Markup Language.
It is the standard language used to create web pages.

HTML tells the browser what content to show
HTML defines structure not design
Every website starts with HTML

HTML does not
Add colors (CSS does that)
Add logic or behavior (JavaScript does that)

Real world analogy

HTML is like the skeleton of the body
CSS is like clothes
JavaScript is like the brain and muscles

Example

<h1>Welcome to My Website</h1>
<p>This is a paragraph.</p>

----------------------------------------------------------------------------------------------------------------------

HTML HISTORY

HTML was created in 1991 by Tim Berners Lee.
Initially it was used only for sharing documents.
Later it evolved to support images forms and videos.

HTML5 released in 2014 introduced
Audio and video support
Semantic tags
Better performance

Why history matters

Interviewers often ask the difference between HTML and HTML5.

----------------------------------------------------------------------------------------------------------------------

HTML STRUCTURE

Every HTML page follows a fixed structure.

Basic structure

<!DOCTYPE html>

<html>
<head>
<title>Page Title</title>
</head>
<body>
<h1>Main Heading</h1>
<p>Content goes here</p>
</body>
</html>

Explanation

The browser reads the page from top to bottom.
The head section contains invisible information.
The body section contains visible content.

Without proper structure the browser may behave incorrectly.

----------------------------------------------------------------------------------------------------------------------

HTML DOCUMENT PARTS

DOCTYPE DECLARATION

The doctype declaration tells the browser that the document is HTML5.
It helps prevent browser rendering errors.

Example

<!DOCTYPE html>

----------------------------------------------------------------------------------------------------------------------

HTML TAG

The html tag is the root element of the page.
It wraps the entire HTML document.

Example

<html>
</html>

----------------------------------------------------------------------------------------------------------------------

HEAD TAG

The head tag contains metadata.
Its content is not visible to users.

Used for
Page title
SEO information
Linking CSS and JavaScript files

Example

<head>
<title>My Website</title>
</head>

----------------------------------------------------------------------------------------------------------------------

BODY TAG

The body tag contains all visible content.

Example

<body>
<h1>Hello World</h1>
<p>This text is visible</p>
</body>

Think of it like this

The head is the brain planning things
The body is the physical appearance

Why Visual Studio Code is recommended

HTML does not require installation.

HTML files must use the following extensions

.html
.htm

Examples

index.html
about.html
contact.html

The index.html file is the default homepage.

The browser recognizes HTML files using the .html extension.

Comments are notes written for developers.
