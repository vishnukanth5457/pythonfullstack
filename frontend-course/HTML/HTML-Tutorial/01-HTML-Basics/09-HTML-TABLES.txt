----------------------------------------------------------------------------------------------------------------------

HTML TABLES

Tables are used to display tabular data (rows and columns).
Tables should not be used for page layout.

----------------------------------------------------------------------------------------------------------------------

BIG PICTURE

Rows represent horizontal data
Columns represent vertical data
A table is similar to a spreadsheet

Analogy
An HTML table is like an Excel sheet

----------------------------------------------------------------------------------------------------------------------

TABLE STRUCTURE TAGS

----------------------------------------------------------------------------------------------------------------------

TABLE TAG

The table tag is the container for the entire table.

Example

<table>
</table>

Without the table tag, other table elements will not work.

----------------------------------------------------------------------------------------------------------------------

TR TAG

The tr tag defines a single table row.

Example

<tr>
</tr>

----------------------------------------------------------------------------------------------------------------------

TH TAG

The th tag defines a table header cell.

Characteristics
Column headings
Bold and center aligned by default
Has semantic meaning

Example

<th>Name</th>
<th>Age</th>

Screen readers treat header cells specially.

----------------------------------------------------------------------------------------------------------------------

TD TAG

The td tag defines a normal table data cell.

Example

<td>Vishnukanth</td>
<td>22</td>

----------------------------------------------------------------------------------------------------------------------

BASIC TABLE EXAMPLE

<table border="1">
<tr>
<th>Name</th>
<th>Role</th>
</tr>
<tr>
<td>Vishnu</td>
<td>Developer</td>
</tr>
</table>

----------------------------------------------------------------------------------------------------------------------

TABLE METADATA TAGS

----------------------------------------------------------------------------------------------------------------------

CAPTION TAG

The caption tag provides a title for the table.

Example

<caption>Employee Details</caption>

The caption appears at the top of the table.
It improves accessibility.

----------------------------------------------------------------------------------------------------------------------

THEAD TAG

The thead tag groups header rows.

Example

<thead>
<tr>
<th>ID</th>
<th>Name</th>
</tr>
</thead>

----------------------------------------------------------------------------------------------------------------------

TBODY TAG

The tbody tag contains the main table data.

Example

<tbody>
<tr>
<td>1</td>
<td>Alice</td>
</tr>
</tbody>

----------------------------------------------------------------------------------------------------------------------

TFOOT TAG

The tfoot tag is used for totals or summary information.

Example

<tfoot>
<tr>
<td colspan="2">Total Employees: 10</td>
</tr>
</tfoot>

The footer is rendered at the bottom.
The browser may load it first for performance.

----------------------------------------------------------------------------------------------------------------------

ROW AND COLUMN SPANNING

----------------------------------------------------------------------------------------------------------------------

ROWSPAN

Rowspan is used to merge rows vertically.

Example

<td rowspan="2">Admin</td>

----------------------------------------------------------------------------------------------------------------------

COLSPAN

Colspan is used to merge columns horizontally.

Example

<td colspan="2">Total</td>
