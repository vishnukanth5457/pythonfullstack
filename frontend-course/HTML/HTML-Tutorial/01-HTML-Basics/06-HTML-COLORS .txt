TYPES OF HTML COLORS

---

COLOR NAMES

HTML supports predefined color names.

Common color names include
red
blue
green
black
white
yellow
orange
purple
gray

Example

<p style="color: red;">Red text</p>

<p style="background-color: blue; color: white;">

Color names are easy to use

Analogy

---

HEX COLORS

HEX colors are the most commonly used format.

Format

#RRGGBB

RR represents red
GG represents green
BB represents blue

Values range from 00 to FF

Example

<p style="color: #ff5733;">HEX colored text</p>

Common HEX colors

Black  #000000
White  #ffffff
Red    #ff0000
Green  #00ff00
Blue   #0000ff

HEX values are case insensitive
HEX is the industry standard for web design

Analogy
HEX colors are like precise paint codes

---

SHORT HEX FORMAT

Short HEX form can be used when both digits are the same.

Example

<p style="color: #f00;">Red</p>

This is equivalent to

#ff0000

---

RGB COLORS

RGB stands for Red Green Blue.

Format

rgb(red, green, blue)

Each value ranges from 0 to 255

Example

<p style="color: rgb(255, 87, 51);">
RGB colored text
</p>

RGB is easy to understand
It is useful when colors are generated dynamically

---

RGBA COLORS

RGBA is RGB with an alpha value.

Alpha controls transparency.

Format

rgba(red, green, blue, alpha)

Alpha values range from
0 for fully transparent
1 for fully solid

Example

<p style="background-color: rgba(255, 0, 0, 0.5);">
Transparent red background
</p>

RGBA is commonly used for overlays and modals

Analogy
RGBA is like colored glass

---

HSL COLORS

HSL stands for Hue Saturation and Lightness.

Format

hsl(hue, saturation, lightness)

Hue ranges from 0 to 360
Saturation controls color intensity
Lightness controls brightness

Example

<p style="color: hsl(10, 100%, 60%);">
HSL colored text
</p>

HSL makes brightness adjustments easier

---

HSLA COLORS

HSLA is HSL with alpha transparency.

Format

hsla(hue, saturation, lightness, alpha)

Example

<p style="background-color: hsla(200, 100%, 50%, 0.4);">
Transparent HSL background
</p>
