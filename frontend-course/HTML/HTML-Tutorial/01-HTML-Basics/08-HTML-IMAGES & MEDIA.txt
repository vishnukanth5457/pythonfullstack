PART A IMAGES

----------------------------------------------------------------------------------------------------------------------

IMG TAG

The img tag is used to display images.

Basic syntax

<img src="image.jpg" alt="Description">

The img tag is a void element
It does not have a closing tag
It always requires src and alt attributes

----------------------------------------------------------------------------------------------------------------------

SRC ATTRIBUTE

The src attribute specifies the image path.

Example using relative path

<img src="images/car.jpg" alt="Red car">

Example using absolute path

<img src="https://example.com/car.jpg" alt="Car">

If the src path is broken the browser shows a broken image icon

----------------------------------------------------------------------------------------------------------------------

ALT ATTRIBUTE

The alt attribute provides alternative text.

Alt text is shown when
The image fails to load
A screen reader reads the page

Example

<img src="car.jpg" alt="Red sports car parked on road">

Alt improves SEO and accessibility

Analogy
Alt text is a spoken description of the image

----------------------------------------------------------------------------------------------------------------------

WIDTH AND HEIGHT ATTRIBUTES

Used to control the image size.

Example

<img src="car.jpg" alt="Car" width="300" height="200">

Setting width and height prevents layout shifting
It improves page performance
Images should not be stretched unnaturally

----------------------------------------------------------------------------------------------------------------------

IMAGE FORMATS

JPG or JPEG
Used for photographs

PNG
Used for images with transparency

SVG
Used for icons and logos
Scales without losing quality

WEBP
High quality with small file size
Best for performance

----------------------------------------------------------------------------------------------------------------------

IMAGE BEST PRACTICE EXAMPLE

<img 
src="car.webp"
alt="Red car on highway"
width="400"
loading="lazy">

Lazy loading improves page speed

----------------------------------------------------------------------------------------------------------------------

PART B AUDIO

----------------------------------------------------------------------------------------------------------------------

AUDIO TAG

The audio tag is used to play sound files.

Basic example

<audio src="song.mp3" controls></audio>

The controls attribute provides play pause and volume controls

----------------------------------------------------------------------------------------------------------------------

SOURCE TAG FOR AUDIO

Used to provide multiple audio formats.

<audio controls>
<source src="song.mp3" type="audio/mpeg">
<source src="song.ogg" type="audio/ogg">
Your browser does not support audio.
</audio>

The browser selects the best supported format
