PART A QUOTATION AND REFERENCE TAGS

----------------------------------------------------------------------------------------------------------------------

BLOCKQUOTE TAG

The blockquote tag is used for long quotations taken from another source.

Example

<blockquote>
The future belongs to those who learn more skills and combine them in creative ways.
</blockquote>

The browser adds indentation automatically
This is a semantic and SEO friendly tag

Analogy
A quoted paragraph in a book

----------------------------------------------------------------------------------------------------------------------

Q TAG

The q tag is used for short quotations inside a sentence.

Example

<p>
HTML is <q>the structure of the web</q>.
</p>

The browser automatically adds quotation marks
Used for inline quotes

Analogy
A quotation inside a sentence

----------------------------------------------------------------------------------------------------------------------

ABBR TAG

The abbr tag is used for abbreviations.

It shows the full meaning when hovered.

Example

<p>
<abbr title="HyperText Markup Language">HTML</abbr> is used for web pages.
</p>

It improves accessibility
Screen readers read the full form

Analogy
A short name with explanation

----------------------------------------------------------------------------------------------------------------------

ADDRESS TAG

The address tag is used for author or organization contact information.

Example

<address>
Written by Vishnukanth<br>
Hyderabad India<br>
Email example@gmail.com
</address>

The text is usually italic by default
It should not be used for random postal addresses

----------------------------------------------------------------------------------------------------------------------

CITE TAG

The cite tag is used to reference the title of a work.

It can reference books movies websites or research papers.

Example

<p>
<cite>HTML Complete Guide</cite> by W3C
</p>

The cite tag is used only for titles
It should not be used for person names

----------------------------------------------------------------------------------------------------------------------

BDO TAG

The bdo tag is used to override text direction.

Example

<bdo dir="rtl">Hello World</bdo>

rtl means right to left
Used for languages like Arabic and Hebrew

----------------------------------------------------------------------------------------------------------------------

PART B CODE AND PROGRAMMING TAGS

----------------------------------------------------------------------------------------------------------------------

CODE TAG

The code tag is used for short inline code snippets.

Example

<p>
Use <code>console.log()</code> to print output.
</p>

Text appears in monospace font
Displayed inline with text

----------------------------------------------------------------------------------------------------------------------

PRE TAG

The pre tag preserves formatting.

It preserves spaces
It preserves line breaks
It preserves indentation

Example

<pre>
function add(a, b) {
  return a + b;
}
</pre>

Often used together with the code tag

----------------------------------------------------------------------------------------------------------------------

KBD TAG

The kbd tag is used to represent keyboard input.

Example

<p>
Press <kbd>Ctrl</kbd> + <kbd>C</kbd> to copy.
</p>

Used for keyboard shortcuts and instructions

----------------------------------------------------------------------------------------------------------------------

SAMP TAG

The samp tag is used to represent sample program output.

Example

<p>
Output <samp>Hello World</samp>
</p>

It represents results not code

----------------------------------------------------------------------------------------------------------------------

VAR TAG

The var tag is used to represent variables in math or programming.

Example

<p>
Area = œÄ √ó <var>r</var><sup>2</sup>
</p>

It provides semantic meaning for variables
